# HTML & CSS Mini Projects

This repository contains a collection of mini projects developed using **HTML5 and CSS3**. These projects demonstrate basic web development concepts such as form design, page layout, styling, and simple web simulations.

## Projects Included

### 1. Event Registration Form

A simple web form that allows users to register for an event by entering details such as name, email, and other required information.

### 2. Store Page Design

A basic store or product page layout created using HTML and CSS that demonstrates how an online shopping page can be structured and styled.

### 3. Game Simulation

A simple game simulation interface designed using HTML and CSS to showcase layout and visual design techniques.

### 4. Digital Clock Simulation

A digital clock interface that visually displays the current time and demonstrates basic front-end design concepts.

## Technologies Used

* HTML5
* CSS3
